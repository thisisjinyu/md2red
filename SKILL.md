# md2red

> 本文件是整套技能的**路由入口**。本身只负责「什么时候该读哪份文档」，不重复具体规则。具体规则全部在 `docs/` 里。

## 这是什么

把一篇 Markdown 文稿 → 拆成一幅幅小红书图文卡片。

- **画布**：固定 `1080×1440`（3:4）。
- **一套图文**：封面 1 张 + 弹性正文 N 张 + 尾页 1 张。
- **三套主题**：B 清新薄荷 / C 酸性霓虹 / D 美拉德×莫兰迪，一套图文只用一个。
- **输出**：HTML 卡片 → 浏览器打印成 PDF → 拆成 PNG。

## 工作流

```
Markdown 文稿
   ↓  拆解（AI 拆块 + 手动 --- 分页）
轻标记语义块（标题/要点/数据/出处/CTA）
   ↓  映射（见 03-content-mapping）
卡片 HTML（套用 04-layouts 配方 + 05-components 控件）
   ↓  选主题（见 02-palettes）
   ↓  浏览器 Cmd+P 导出 PDF（1080×1440）
   ↓  pdftoppm / magick 拆成图片
PNG 图文，发布
```

在线生成器：仓库根的 `index.html`（可部署到 GitHub Pages）已把「输入 md → 实时预览卡片 → 导出」跑通。

## 输入：轻标记 Markdown 方言（速查）

详规则见 `03-content-mapping.md`。速查：

| 写法 | 变成 |
| --- | --- |
| `# 标题` | 封面主标（首个数字自动放大）|
| `话题：` | 封面话题标签 |
| `导读：` / 首段 | 封面导语 |
| `## 标题` | 正文卡主标（自动编号）|
| 普通段落 | 正文要点 `.point` |
| `**42% ↑** 说明…` | 数据块 `.stat` |
| `> 出处` | 数据出处 `.src` |
| `---` | 续卡（同标题 + 「 ·续」）|
| `结尾：` | 尾页收尾大标 |
| `号召：` / `CTA：` | 尾页 CTA |
| `//` | 强制换行 `<br>` |

## 硬规则（摘要）

完整 10 条见 `05-components.md` §7。最容易犯的：

1. 卡片永远 `1080×1440`，`flex` 撑满，禁 `vw/vh`。
2. 内容填满 ≥ 75% 画布，正文卡用 `.spacer` 把数据块压底。
3. 一张正文卡最多一个 `.stat`。
4. 文字最小 `22px`，塞不下删内容不缩字号。
5. 光晕/幽灵数字/渐变描字是 **C 主题专属**。
6. 不编造数据，未核标「示例数据 · 待核」。
