# md2red

把 Markdown 内容转成小红书图文卡片的模板系统。

## 这是什么

md2red 收敛一套「内容 → 小红书轮播图」的设计规则与可复用模板。目标：给定一篇结构化内容（标题 + 若干要点 + 数据 + 出处），稳定产出风格统一、移动端可读的封面 / 正文 / 尾页卡片。

组织方式参考 guizang social-card skill（规则文档 + 模板 + 校验），但配色、版式、文案策略按 Eat This 资料库的实际内容重做。

## 目录结构

- `docs/` — 设计规则（配色体系、版式纪律、类型刻度、间距 token、卡片分工）
- `templates/` — 可复用卡片模板（按风格分：swiss / editorial）
- `samples/` — 用真实选题渲染的效果图

## 风格

- **Swiss 国际主义**（清新科普 / 数据向）：无衡线、网格、单一锚点色、越大越细、无圆角无阴影。
- **Editorial 杂志风**（叙事 / 人文向，后续）：衡线标题、纸张氛围、留白。

## 配色系列

- **B 清新薄荷**（健康饮食 HE）：底 雾蓝白，锚点 信任蓝 `#2D7FF9`
- **C 酸性·多巴胺霾虹**：底 炭黑，锚点 多色霾虹
- **D 美拉德×莫兰迪**：底 暖奶油，锚点 赤陶 / 鼠尾草
