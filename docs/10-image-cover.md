# 10 · 主图封面：压图与避脸（Image Cover）

> 范围裁剪：对齐 guizang `references/image-overlay.md` + `layout-recipes.md` M16 / S08，**只取「主图封面」一种用法**。
> 正文配图证据页、截图网格、UI mock、地图组件**不在本系统能力内**（见 `08-recipe-catalog.md` 🚫）。
> 默认仍是**文字封面**（`cover` 组件）。主图封面是 lifestyle / 食材成图 / 真实场景照的**可选开口**，不是默认。

---

## 何时才用主图封面

仅当用户**有一张足够好的照片**且内容是图片驱动（旅行 / 户外 / 自家成菜 / 家居 / 真实情绪场景）。新闻 / 教程 / 数据 / 测评类封面是「钩子句」，用文字 `cover`，不要硬塞照片。

**选图门槛（两测必须都过，否则退回文字 `cover`）：**

1. **静区测试**：照片有 ≥30% 画布的低细节 / 失焦 / 均匀带（虚化背景、雾、静水、纯天空）——标题落这里。
2. **光线测试**：氛围 / 克制光（阴天、晨雾、黄金时刻、林下、暮色剪影、胶片柔光）。**拒绝**高饱和正午、机顶闪光、游客打卡照。

只过一个 → 退文字 `cover`（照片塞进小框 `.frame-img` 而非整屏）。一个都不过 → 换图。**绝不靠蒙版救烂照片。**

---

## Rule 1 · 先选图，蒙版只在选图失败时才上

默认答案是**无蒙版**。Kinfolk / Cereal / Monocle 这类封面几乎从不在主图上盖整屏黑色渐变——靠的是选图 + 构图让标题自然落在静区。

按序试，**第一步过就停**：

1. **选图**（主杠杆）：过上面两测。
2. **无蒙版构图**：标题放进静区 → 跑缩略图测试（第 4 步）→ 过就交付。多数选对的照片在此通过。
3. **局部、图调色蒙版**（仅当第 2 步对比度不过）：
   - **局部**，非整屏：`radial-gradient` 罩住标题区，或单侧线性渐变向外渐隐到透明。**绝不** `inset:0` 整屏均匀压。
   - **图调色**，非黑：取照片里已有的暗调（林地 `#1a2818` / 暮色 `#4a2638` / 雪 `#2a3438`）。纯黑 `#000` 读作「标注层」。
   - **柔**：标题区峰值 α 0.15-0.30，向外到 0。要 >0.40 才看清 = 照片没过第 1 步，回去换图。
4. **缩略图对比度测试**（永远做）：渲染后缩到 360px 宽看，标题描边不眯眼可读。

```css
/* 局部、图调色蒙版示例：标题在左下 */
.cover-photo::after{
  content:"";position:absolute;inset:0;pointer-events:none;
  background:radial-gradient(60% 45% at 25% 80%,
    rgba(26,40,24,.28) 0%, rgba(26,40,24,0) 100%);
}
```

**禁**：整屏竖向黑渐变（game key-art 做法）/ 纯黑蒙版 / 文字底加黑白实色块 / `img{opacity:.6}` / 没认真选图就直接上蒙版。

---

## Rule 2 · 文字避开主体 / 脸

### 主体地图（先看图，再排版）

排标题前先观察并把答案写成 HTML 注释贴在主图块旁：

```html
<!-- 主体地图（封面 · 稻城晨湖）：
     主焦点：雪峰 50% x 32% y（中带）
     安全文字区：顶带 0-12% y（低细节天空）、底带 72-95% y（静水倒影）
     静区测试：过 —— 底 30% 为均匀水面
     光线测试：过 —— 晨间阴天柔光 -->
```

### 安全区放置（按优先级）

1. **上 + 下**：顶 kicker + 底标题。九成主图封面这么做。
2. **单侧**：主体占一竖栏 → 文字放对侧栏（对侧需 ≥40% 干净宽）。
3. **对角**：主体在一角、对角真空时才用。

**绝不**把大标压过脸。90px 中文标题切过眼睛即使加蒙版也是破坏性的。

### `object-position` 按主体位置选（别留默认）

| 原图主体位置 | 3:4 裁切建议 |
| --- | --- |
| 脸/焦点在上三分 | `center 25%`（抬高脸，给底部留标题位）|
| 脸/焦点在中三分 | `center center`（默认；文字上下走）|
| 脸/焦点在下三分 | `center 70%`（压低脸，顶部留标题位）|
| 宽景 / 无单一主体 | `center 35%`（略上偏保住地平线）|
| 竖向全身像 | `center top` |

### 排版克制（编辑感而非「图配字」）

- **不对称**：标题偏一侧/一角，比居中更自信（Kinfolk 习惯）。
- **留白慷慨**：标题占画布 ≤40%，照片和静区在做设计。
- **标题只落一个静区**（上+下两段可以，三段或绕主体不行）。
- 主图封面字色用**纸白** `#f5f1e8`，不用纯白；标题用衬线/我们的主标字重（细，**不加粗**）；不做整屏蒙版「以防万一」。

---

## 主体撑满全画布、无安全区时

不要硬塞文字。改：① 照片缩进定义好的小框（`.frame-img`）+ 周围留白与文字；② 双图拼版（一张紧主体 + 一张留白），标题压在留白图上；③ **文字完全移出照片**——标题在上、照片在下。
